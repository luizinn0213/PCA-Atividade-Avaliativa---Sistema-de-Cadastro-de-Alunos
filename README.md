Esse código tem por finalidade auxiliar no cadastro e visualização dos alunos cadastrados. O código permite guardar informações de determinados alunos, assim como pesquisá-los por nome ou por matrículo (gerada automaticamente pelo código). Sua utilização é simples e intuítiva onde é preciso digitar pequenos parâmetros para o fluxo do sistema rodar tranquilamente.

Caso não tenho no seu VSCode, é precisa instalar um Python Interpreter para que o programa roda normalmente.
